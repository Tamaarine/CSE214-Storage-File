HOMEWORK - FALL 2019

Programming Assignment 5 -  GRADING KEY

1. (17 points) Documentation and coding style:

  • Name, ID number and recitation at the top of ALL java files [2 points]
  • Precise and correct specifications [5 points]
  • Indentation [3 points]
  • Variable names [2 points]
  • Javadoc Style documentation *(see NOTE). [5 points]

NOTE 1: You only need to submit your .java files. We will use the javadoc *.java command to create the javadoc files.

NOTE 2: YOUR ASSIGNMENT WILL NOT BE GRADED if you submit only *.class files.

2. (30 points) Source code:
  • Implementation done as specified. [8 points]
  • Throws an exception back to the calling method(s) if a precondition is violated. [7 points]
  • Programs compiles without any errors. [15 points]
3. (53 points) Program correctness:
  • Menu operations work correctly:
    • PC - Create Plant Child [8 points]
    • AC - Create Animal Child [8 points]
    • RC - Remove Child [8 points]
    • P - Print Out Cursor's Prey [4 points]
    • C - Print Out Food Chain [5 points]
    • F - Print Out Food Pyramid at Cursor [7 points]
    • LP - Print Out Plants Supporting Cursor [6 points]
    • R - Reset Cursor To Root [2 points]
    • M - Move Cursor To Child [4 points]
    • Q - Quit [1 point]
