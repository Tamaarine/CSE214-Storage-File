HOMEWORK - FALL 2019

Programming Assignment 3 -  GRADING KEY

1. (17 points) Documentation and coding style:

  • Name, ID number and recitation at the top of ALL java files [2 points]
  • Precise and correct specifications [5 points]
  • Indentation [3 points]
  • Variable names [2 points]
  • Javadoc Style documentation *(see NOTE). [5 points]

NOTE 1: You only need to submit your .java files. We will use the javadoc *.java command to create the javadoc files.

NOTE 2: YOUR ASSIGNMENT WILL NOT BE GRADED if you submit only *.class files.

2. (38 points) Source code:
  • Implementation done as specified.
    • Push and pop off blocks correctly when it is required to do so. [8 points]
    • Read in text until a null string is encountered (as opposed to an empty string). [5 points]
    • Implement Stack class or use the Java Stack class correctly, not calling any Vector methods that were inherited, but not listed in the Stack API (using clone is OK). [10 points]
  • Programs compiles without any errors. [15 points]
3. (45 points) Program correctness:
  • Compiles without error (no partial credit)
  • Program handles all printing request cases correctly such as:
    • Variable is correctly printed if it is found inside the current block. [9 points]
    • Variable is correctly printed if it is located in a block containing the current block. [9 points]
    • The most recent version of a variable is printed. [6 points]
    • An appropriate error message is printed if a variable name cannot be found. [4 points]
    • All variables in the current block are correctly printed. [9 points]
    • An appropriate message is printed if there are no variables in the current block. [4 points]
    • All variables without initial values are assumed to be equal to zero. [4 points]
