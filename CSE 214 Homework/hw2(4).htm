This program will only play sound files encoded with the wav format. You can find a selection of music and sounds which are public domain or creative commons in many places on the web. Links to some of these will be provided below.   These songs will be characterized by an artist, album, song name, and song length. The playlist is able to add the songs, remove them, shuffle them, and play them.

You should have getters and setters for each variable as well as a default constructor and possibly any other constructor you see fit.

●       public SongNode - constructor (you may include a constructor with parameters)

●    Song data The song at this location within the list

●       Getter and setter methods for each member variable

USING javax.sound FOR AUDIO FILES:

NOTE: This method of playing audio files will only work for wav files! It will not work for mp3 files!

 

You can import javax.sound to play the audio files.

 

// these three should be at the type of your code as per usual with importing

import javax.sound.sampled.AudioInputStream;

import javax.sound.sampled.AudioSystem;

import javax.sound.sampled.Clip;

 

// this will load and play a wav file

try {

AudioInputStream AIS = AudioSystem.getAudioInputStream(

this.getClass().getResource("name_of_file_in_your_current_working_directory.wav"));

  

// starting the clip

Clip c = AudioSystem.getClip();

            c.open(AIS);   

            c.start();

}

catch (Exception ex) {}

 

This will start the song playing in the background. You can find more information about the javax.sound.sampled library here.
